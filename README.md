# Проект: Продвинутая семантическая дедупликация новостей (NLP)

Масштабируемый NLP-конвейер для автоматического сбора, очистки и жесткой семантической дедупликации потоков новостей с использованием локальных нейросетей. Позволяет сжимать огромные объемы данных, удаляя смысловые повторы, рерайт и технический мусор.

## 🚀 Финальные результаты (Production)
*   **Масштаб**: 15 400 реальных новостных статей (Английский и Русский) с 60+ мировых изданий.
*   **Метрика качества (Adjusted Rand Index)**: **0.9925** (Цель была >0.97).
*   **Precision**: 96.68% | **Recall**: 98.90% | **F1-Score**: 97.78%
*   **Синтетический тест**: Алгоритм безупречно нашел 989 из 1000 искусственных дубликатов (перестановка предложений, удаление слов) в массиве из 14 400 реальных новостей.

## ⚙️ Архитектура Пайплайна

Пайплайн (`news_dedup_pipeline.py`) работает локально на видеокарте (без вызовов платных API) и состоит из 4 этапов:

1.  **MinHash Exact Pre-filter (Предфильтр)**
    Мгновенно находит 100% точные или технические дубликаты (Jaccard > 0.99). Снижает вычислительную нагрузку на нейросети.
2.  **Sliding Window Similarity Search (E5 Embeddings)**
    *   **Модель**: `intfloat/multilingual-e5-base`
    *   Генерирует семантические векторы (эмбеддинги) для всех статей.
    *   Ищет совпадения только в пределах скользящего временного окна (по умолчанию 2 дня). Это исключает сравнение новостей из разных эпох и ускоряет FAISS/Torch поиск в сотни раз.
3.  **Cross-Encoder Reranker (Тяжелая проверка)**
    *   **Модель**: `BAAI/bge-reranker-v2-m3`
    *   Берет "подозрительные" пары из предыдущего шага и перепроверяет их, передавая оба текста в нейросеть одновременно. Это отсеивает 99% ложных срабатываний (False Positives).
4.  **Louvain Graph Clustering (Графовая сборка)**
    Строит граф, где узлы — статьи, а ребра — подтвержденные Cross-Encoder'ом связи. Алгоритм выделяет сообщества (кластеры) и выбирает самую старую статью в качестве "первоисточника" (Canonical Article).

## 🛠 Установка и Запуск

1.  **Установка зависимостей:**
    ```bash
    pip install -r requirements.txt
    ```
2.  **Сбор данных (Парсинг 15 000 статей):**
    ```bash
    python scrape_10k_news_phase2.py
    ```
3.  **Запуск дедупликации:**
    ```bash
    python news_dedup_pipeline.py --input data/news_15k_ready.json --cross_threshold 0.60
    ```
4.  **Генерация синтетических тестов (опционально):**
    ```bash
    python generate_nlp_duplicates.py --input data/news_dump_cleaned.json --output data/synthetic.json --limit 1000
    python news_dedup_pipeline.py --input data/news_dump_cleaned.json --dup_input data/synthetic.json --output data/news_dump_polluted.json --clusters_output data/clusters_polluted.json
    ```

## 📁 Структура Данных
*   `data/news_dump_cleaned.json` — Идеальный, кристально чистый набор уникальных каноничных сюжетов.
*   `data/clusters_output.json` — Карта кластеров, связывающая каноничные ID со всеми найденными копиями.
